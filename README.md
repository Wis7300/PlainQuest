## LANCEMENT DU PROJET

### ENVIRONNEMENT :

testé sous Windows:

- compatible avec python 3.12 et 3.13

- non compatible avec python 3.14 (bibliothèque `arcade`)

***

  

### INSTALLATION :

- exécuter cette commande dans le terminal pour installer les bibliothèques requises:
`pip install -r requirements.txt`

***

  

### LANCEMENT :
- double clique sur le fichier main.py
- Si cela ne marche pas : double clique sur le fichier lancer_jeu.bat
- SI cela ne  marche vraiment pas : exécuter cette commande dans un terminal : `python main.py`
